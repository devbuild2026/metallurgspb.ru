# РусМет — Структура проекта и инструкция

## Стек технологий

- **Next.js 14** (App Router) — фреймворк
- **TypeScript** — язык разработки
- **Tailwind CSS** — стили
- **React** — UI-библиотека

---

## Как запустить

```bash
npm install          # установить зависимости (один раз)
npm run dev          # запустить локальный сервер → http://localhost:3000
npm run build        # собрать продакшн-версию
```

## Как редактировать контент

### Добавить / изменить товар
→ `data/products.ts`

Каждый товар — объект:
```ts
{
  id: 42,
  name: "Арматура рифленая А-3 500С 16 мм",
  slug: "armatura-riflenaya-16mm",   // уникальный URL-идентификатор
  categorySlug: "riflenaya-a3-500s", // к какой подкатегории относится
  price: 45.50,                       // цена за единицу
  unit: "м.п.",                       // единица измерения
  size: "11,7 м"                      // размер/вес
}
```

### Добавить / изменить категорию
→ `data/categories.ts`

Структура двухуровневая: **группа** (например, «Арматура») → **подкатегории** (рифлёная, гладкая, композитная).

### Привязать изображение к категории
→ `data/categoryImages.ts`

Добавить строку:
```ts
'slug-kategorii': '/images/categories/папка/файл.png',
```

### Добавить изображение категории
1. Положить файл в `public/images/categories/название_папки/`
2. Прописать путь в `data/categoryImages.ts`

### Изменить SEO-текст или FAQ категории
→ `data/categorySeo.ts`

Каждая группа содержит:
- `text` — вводный абзац
- `sections[]` — разделы с подзаголовками
- `faq[]` — вопросы и ответы

### Изменить отзывы
→ `components/ReviewsBlock.tsx` — массив `reviews` в начале файла

### Изменить логотип
→ `public/images/logo/` — заменить `main_logo.png`
→ `components/Header.tsx` и `components/Footer.tsx` — строка `src="/images/logo/..."`

### Изменить фон главного экрана
→ `public/images/background/` — заменить `background1.png`

### Изменить контакты (телефон, email, адрес)
Контакты встречаются в нескольких местах — менять во всех:
- `components/Header.tsx`
- `components/Footer.tsx`
- `app/contacts/page.tsx`
- `app/delivery/page.tsx`

## SEO-настройки

| Что | Где |
|-----|-----|
| Title + Description главной | `app/layout.tsx` → `metadata` |
| Title + Description каталога | `app/catalog/page.tsx` → `metadata` |
| Title + Description категорий | `app/catalog/[slug]/page.tsx` → `generateMetadata()` |
| Title + Description товаров | `app/product/[slug]/page.tsx` → `generateMetadata()` |
| SEO-тексты и FAQ категорий | `data/categorySeo.ts` |
| robots.txt | `app/robots.ts` (автогенерация) |
| sitemap.xml | `app/sitemap.ts` (автогенерация) |
| Микроразметка Organization | `app/layout.tsx` → `<Script id="schema-organization">` |
| Микроразметка Product | `app/product/[slug]/page.tsx` → `productSchema` |

> ⚠️ После смены домена заменить `https://rusmet.ru` в `app/sitemap.ts`, `app/robots.ts` и во всех `alternates.canonical`.

## Логика корзины

- Состояние хранится в `localStorage` через `CartProvider`
- Добавление товара: компонент `OrderControls` (страница товара) или кнопка «В корзину» в таблице каталога
- Попап корзины: `CartPopup` — появляется при добавлении товара
- Страница корзины: `app/cart/page.tsx`
- Оформление: `app/checkout/page.tsx`

## Калькулятор доставки

→ `components/DeliveryCalculator.tsx`

Тарифы задаются в массиве `truckTypes`:
```ts
{ id: 'small', label: 'Малотоннажная (до 3 т)', base: 4000, perKm: 35, perKmFar: 50 }
```
- `base` — базовая стоимость (₽)
- `perKm` — цена за км (до 50 км в одну сторону)
- `perKmFar` — цена за км сверх 100 км (туда-обратно)
